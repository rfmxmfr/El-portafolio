Deploy to Netlify

This page provides instructions for deploying your Fashion Portfolio to Netlify.

Deploy to Netlify

Step 1: Click the button above

This will take you to Netlify and connect to your GitHub repository.

Step 2: Configure the build settings

Use these settings:

  • Build command: npm run build
  • Publish directory: dist

Step 3: Advanced build settings

Add these environment variables:

  • Key: NODE_VERSION, Value: 18
  • Key: NPM_FLAGS, Value: --legacy-peer-deps

Step 4: Deploy!

Click the deploy button and wait for your site to build.

Alternatively, you can build locally and deploy the dist folder directly:

  1. Run npm run build to generate the dist folder
  2. Go to netlify.com/drop
  3. Drag and drop the dist folder onto the page
